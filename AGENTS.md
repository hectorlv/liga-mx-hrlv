# AGENTS
- Para compilar el proyecto usa `corepack pnpm run build`.
- Si el servidor no está levantado, usa `corepack pnpm run start:build:open` para compilar, levantar el servidor y poder abrir en el navegador.
- Después de un cambio, si el servidor ya está levantado solo hace falta volver a compilar y esperar para actualizar en el navegador.
